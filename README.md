# 🎲 Dice Rolling Game

A simple Python dice rolling game built as a beginner project.

## Roadmap

1. Build an infinite loop so the game keeps running.
2. Ask the user whether they want to roll the dice.
3. If the user enters `y`, generate two random numbers and print them.
4. If the user enters `n`, end the game with a goodbye message.
5. If the user enters anything else, show an error message.

## Features

* Roll two dice randomly.
* Continue playing as many times as you want.
* Accepts both uppercase and lowercase input.
* Displays an error message for invalid choices.
* Exit the game anytime.

## Concepts Used

* Python loops (`while`)
* Conditional statements (`if`, `elif`, `else`)
* Random module
* User input
* Functions from the `random` library
* String methods (`lower()`)

## How to Run

```bash
python dice_rolling_game.py
```

## Example

```text
Roll the dice? (y/n): y
You rolled: (4, 6)

Roll the dice? (y/n): n
Thanks for playing!
```

This project helped me practice Python fundamentals such as loops, conditions, user input, and random number generation.
